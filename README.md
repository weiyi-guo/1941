## Keyboard mapping
* W: Pitch Up
* S: Pitch Down
* A: Yaw Left
* D: Yaw Right
* Space: Speed Up(Throttle Up)
* Left Mouse Button: Main Weapon(machine gun)
* Right Mouse Button: Secondary Weapon(rocket launcher)

## Introduction
A physical based TPS Flight-Simulation and shooting game based on UE5 and C++.
